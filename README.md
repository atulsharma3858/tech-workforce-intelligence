# Tech Workforce Intelligence: Layoffs & Hiring Trends (2020–2025)

## Project Overview
This project analyzes workforce trends in the global technology industry using layoffs data from 2020–2025. The goal is to identify patterns in workforce reductions, understand industry-level impacts, and provide insights into economic shifts affecting the technology sector.

## Business Problem
Technology companies experienced significant workforce restructuring during the recent economic slowdown. Decision-makers need data-driven insights to understand:

- Which industries experienced the most layoffs
- How layoffs evolved over time
- Which companies and regions were most affected
- Early indicators of recovery in the tech sector

## Dataset
Layoffs dataset containing information on:

- Company name
- Industry
- Total employees laid off
- Percentage laid off
- Country
- Funding stage
- Date of layoffs
- Funds raised

Source: Public layoffs dataset (Kaggle)
